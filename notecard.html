Z-SCORE (standardized score)

z = (x − μ) / σ

Meaning: how many standard deviations the value x is above (+) or below (−) the mean.

  • z = 0 → exactly at the mean
  • z = +1 → 1 SD above the mean
  • z = −2 → 2 SD below the mean
  • Larger |z| → farther from typical
Example · Z-score x=85, μ=70, σ=10 → z=(85−70)/10 = +1.50 (1½ SD above the mean)
Geometry check · Cylinder d=4, h=6 → r=2 → V=3.14×2²×6 = 75.36

Percentages of standard deviations (empirical rule)

For data that is roughly normal (bell-shaped), the empirical rule tells what percent of the data falls within a certain number of standard deviations of the mean μ. Think: mark μ at the center, then step left/right by σ.

Distance from mean About this % of data In z-score language
within ±1 SD ≈ 68% −1 ≤ z ≤ +1
within ±2 SD ≈ 95% −2 ≤ z ≤ +2
within ±3 SD ≈ 99.7% −3 ≤ z ≤ +3
each side band 0→1 ≈ 34% half of the middle 68%
below μ − 1σ ≈ 16% z < −1 (left tail of middle)
above μ + 3σ ≈ 0.15% z > +3 (one outer tip)

How to use it on a problem

  1. Confirm data is roughly normal.
  2. Write μ and σ. Find the cutoffs: μ±1σ, μ±2σ, μ±3σ.
  3. Match the question interval to a band on the curve.
  4. Read the % (68 / 95 / 99.7) or half-bands (34 / 16 / 0.15).

Worked examples

1 · Within 1 SD → ≈68%
μ=40, σ=5. Between 35 and 45?
35=μ−1σ, 45=μ+1σ → middle → ≈ 68%
2 · Within 2 SD → ≈95%
μ=80, σ=10. Between 60 and 100?
60=μ−2σ, 100=μ+2σ → ≈ 95%
3 · Beyond +3 SD → ≈0.15%
μ=70, σ=5. Older than 85?
85=μ+3σ → one outer tip → ≈ 0.15%

Link to z: “within 1 SD” means −1 ≤ z ≤ +1. For an exact percentile from a specific z (like z=1.2), use the z ↔ percentile table — not the 68-95-99.7 sketch.
